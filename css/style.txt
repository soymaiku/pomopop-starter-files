1. CSS Setup & Global Variables
Global Reset: Sets box-sizing: border-box; for easier layout management.

Typography: Sets global font-family and disables user-select.

CSS Variables (:root):

--pomodoro, --shortBreak, --longBreak (for main theme colors).

--color-text-light, --color-text-medium, --color-bg-light, --color-progress-bar (for general UI consistency).

Body: Applies initial background-color (using --pomodoro) and sets a transition for mode changes.

2. Layout & Structure (Header, Timer, Progress)
.header: Fixed positioning (position: fixed; top: 0; right: 0;) with z-index: 100.

.header-button: Styles for the Settings, Music, and Tasks buttons (white background, padding, hover effects).

progress: Styles the progress bar at the top of the screen (position: fixed; top: 0;).

Includes vendor prefixes (::-webkit-progress-bar, ::-webkit-progress-value) to ensure custom styling works across browsers.

.timer: Main container, centered on the screen using transform: translate(-50%, -50%);.

.clock: Large font styling (font-size: 15rem;) for the countdown numbers.

.task-name-display: Styles the area showing the currently selected task name.

.timer-controls: Uses flex to arrange the main start/pause button and the reset button.

3. Timer Buttons (Mode, Main, Control)
.mode-button: Styles for "Pomodoro," "Short break," and "Long break" buttons.

.mode-button.active defines the white border for the currently selected mode.

.main-button: Styles for the "Start/Pause" button (large, capitalized).

Includes a shadow (box-shadow) to create a 3D effect.

.main-button.active provides the pressed/pause state by moving the button down (transform: translateY(6px);) and removing the shadow.

.control-button: Styles for the "Reset" button (smaller, white background).

.control-button:active also uses transform to simulate a press.

4. Task System
.task-section: Styles the task sidebar (fixed to the left, semi-transparent white background, overflow-y: auto).

.task-input-group: Styles for the input fields and "Add Task" button (uses flex-direction: column).

.add-task-button: Primary button styling using var(--pomodoro) color.

.task-item: Styles for an individual task in the list. Uses border-left for a colorful accent.

.task-item.selected: Highlights the currently selected task using a light blue background and border.

.task-item.completed: Styles completed tasks with reduced opacity and a line-through effect.

.task-complete, .task-edit, .task-delete: Specific color styles for the control buttons (green, blue, red).

5. Modal Styles (Settings & Music)
.modal: Overlay for the settings and music popups (position: fixed;, background-color: rgba(0, 0, 0, 0.4);).

.modal.open is the class used by JavaScript to display the modal (display: block;).

.modal-content: Styles the white box inside the overlay (margin: 10% auto;, max-width).

.close: Styles the close button (&times;).

.form-group: Styles for input labels and fields within the modals.

.save-button: Styling for the save button inside the settings modal.

.music-player, .music-controls, .volume-control: Flex-based layout for the music selection and volume slider.

6. Utility & Animations
.hidden: Utility class to hide elements (display: none !important;).

@keyframes slideDown and @keyframes slideUp: Defines keyframes for the temporary notification pop-up/fade-out animation.

7. Responsive Design (Media Queries)
@media screen and (max-width: 480px) (Mobile):

Significantly reduces the .clock size (to 5rem).

Makes header buttons smaller.

Reduces size and shadow of main timer buttons.

Crucially, repositions .task-section to the bottom of the screen (bottom: 10px;, top: auto;) for better mobile usability.

@media screen and (min-width: 481px) and (max-width: 768px) (Small Tablets):

Adjusts sizes slightly (e.g., .clock to 10rem).

Keeps task section fixed to the bottom.

@media screen and (min-width: 769px) and (max-width: 1024px) (Large Tablets/Small Desktops):

Further size adjustments.

Keeps .task-section at the top-left (top: 80px;).

@media screen and (min-width: 1600px) (Ultra-wide Screens):

Increases the size of .clock and main buttons for large displays.
