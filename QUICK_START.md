# Quick Start

## 1️⃣ Add Firebase Credentials to `.env`

```env
FIREBASE_API_KEY=YOUR_KEY_HERE
FIREBASE_AUTH_DOMAIN=YOUR_DOMAIN_HERE
FIREBASE_PROJECT_ID=YOUR_PROJECT_ID_HERE
FIREBASE_STORAGE_BUCKET=YOUR_BUCKET_HERE
FIREBASE_MESSAGING_SENDER_ID=YOUR_SENDER_ID_HERE
FIREBASE_APP_ID=YOUR_APP_ID_HERE
FIREBASE_MEASUREMENT_ID=YOUR_MEASUREMENT_ID_HERE
```

## 2️⃣ Start Local Server

```bash
npm start
```

## 3️⃣ Open http://localhost:3000/

That's it! Your app should now work locally, same as on Netlify.
