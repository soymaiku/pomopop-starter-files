About

About Pomopop

Pomopop is a productivity tool built for the Nanays in IT (PBSP) program. It helps users manage time through the Pomodoro Technique. This method breaks work into focused 25 minute intervals followed by short breaks. The application includes task tracking, customizable timers, background music, and immersive video backgrounds to support deep focus.
